**Summery**:\
This is a Wallet plugins.\
That customer can deposit to his/her wallet.\
And he can spend his/her wallet.\
This wallet is multi currencies.

**Installing**:\
Normal

**Plugin Configure**:\
Must set Access Key.\

**Add Package:**\
you can go to Baramjk>Wallet Packages menu.

**Deposit By admin:**\
Go to edit customer page. In bottom of page in Wallet section
you can deposit.

**Customer Wallet & Buy new Package:**\
customer can go to his/her profile. and go to wallet menu.\
and he/she can see his wallet an buy new package.
